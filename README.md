# Shader Studio

A local-first, interactive WebGL shader editor built with Next.js.

Create and tune original shader recipes with live controls for style, palette, surface, movement, frame transforms, and cursor interaction. Export a PNG, GLSL source, a React-ready recipe, or a detailed implementation prompt. 

## Run locally

```bash
npm install
npm run dev
```

## Validate and maintain previews

```bash
npm run build
npm run previews
```

The preview utility prepares deterministic capture fixtures for each built-in shader style.
